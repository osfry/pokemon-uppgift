Pokemon-projekt

En interaktiv Pokédex-applikation byggd med React som låter användare utforska de första 151 Pokémon från Generation 1.

Användning

1. Klicka på "Start Pokemon App" på startsidan
2. Välj en Pokémon från dropdown-menyn
3. Klicka "Pick" för att visa Pokémon
4. Använd vänster/höger-pilar för att bläddra mellan Pokémon
5. Skriv `/pokemon/[namn eller id]` i URL:en för direkt åtkomst till den pokemon du vill se
